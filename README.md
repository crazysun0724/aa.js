# aa.js

A lightweight JavaScript utility library focused on writing code naturally.

aa.js was created to reduce boilerplate and make JavaScript feel more comfortable to write.

The goal is not to replace existing frameworks, but to provide a small set of utilities that simplify everyday code while staying close to standard JavaScript.

## Features

* `tag()` – Create HTML elements with flexible arguments.
* `input` – Simple helpers for common input elements.
* `put()` – Append children with a short, chainable API.
* `on()` – Short alias for `addEventListener()`.
* `convertBase()` – Convert values between numeral systems.
* `color()` – Utility functions for color conversion.

## Philosophy

* Reduce unnecessary syntax.
* Keep APIs flexible.
* Prefer readability over cleverness.
* Stay close to native JavaScript.

## Status

This library is currently under active development.
The API may change before version 1.0.
